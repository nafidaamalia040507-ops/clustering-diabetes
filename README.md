# Clustering Analysis of Diabetes Data

Dashboard analisis clustering data diabetes agregat per negara
menggunakan 5 metode clustering:

- K-Means
- Hierarchical Clustering
- Gaussian Mixture Model
- Mean Shift
- Spectral Clustering

## Dataset
Sumber data: World Health Organization (WHO)

## Tools
- Python
- Streamlit
- Scikit-learn
